Instructions

  1. Create the form shown below according to the specifications shown on the "sticky notes". Don't worry about the style of the form until the end of the assignment. The places in the drop-down boxes are:

  2. Ensure that the file "bus.js" is in the same folder as your HTML file and include the following in the <head> section of the HTML file:
    <script src="bus.js"></script>

    The file "bus.js" imports some JavaScript code that does the following:

    • Checks your file to make sure that it follows the specifications.
    • Detects when the "One Way" radio button is checked/unchecked and disables/enables the "returningDate" field.
    • Detects when the "Submit" button is pressed and creates a popup window displaying the information entered on the form.
  3. Open your HTML file in a browser and view the browser's console. (Right-click somewhere on the page, select "inspect", and then in the window that opens, select the "console" tab.)
  4. Take note of any error messages that appear in the console. (The messages all begin with "<< ERROR: ... ") Fix the appropriate error in your HTML so that the error messages no longer appear.
  5. Test the form thoroughly. Make sure the validation for the different input types is functioning as expected.
  6. Style the form to resemble the sample as closely as possible.
    • No inline styles!
    • No tables!
