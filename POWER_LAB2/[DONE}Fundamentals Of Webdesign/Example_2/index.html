P

The <p> tag is the primary tag used for displaying basic text to the user. It displays by default the same way text without a tag shows up.

A

the <a> tag is used to create links to other websites, pages, or elements. The href attribute is required in order for the link to lead anywhere. The text between the opening and closing tag is what is actually displayed to the user.

w3schools article

IMG

The <img> tag is the primary tag used for displaying an image or media on a webpage. The src attribute is required and controls what file will be displayed. The img tag is one of a handful of tags that do not have a closing statement. One of the other attributes that is very good and common practice is the alt attribute. This attribute will control the text a user sees if they hover over the image or the image doesn't load correctly for them.

h1-6

The <h1-6> tags are another element used to display text, however the text is generally larger and bolder. It is meant to provide a series of font changes to make highlighting certain information easier to do. 1 is the largest header, 6 is the smallest.

BR

The <br> tag is used to create a line break in the page, forcing all new elements down into the next row. In some (not necessarily good) cases, <br> tags are used to position elements vertically on the page.
